Publish a web app to Azure with Visual Studio

47 minutes | Module | 8 Units

Overview

Use the publishing features of Visual Studio 2017 to deploy and manage an ASP.NET Core web application hosted on Azure.

In this module, you will:

  • Create an ASP.NET Core web app in Visual Studio 2017
  • Publish the web app to Azure using Visual Studio 2017
  • Update the web app in Visual Studio and publish the changes to Azure
